# Kafka Microservices

A microservices-based setup using **Apache Kafka (KRaft mode)** with Node.js services.

## Tech Stack
- Apache Kafka (KRaft, no ZooKeeper)
- KafkaJS
- Node.js
- Docker & Docker Compose

## Services
- **Kafka Cluster**: 3-broker KRaft setup
- **Kafka Admin Service**: Topic management
- **Other Services**: order, Payment, Email, analytics (event-driven)
